# 更新日志

本文件记录了项目的所有重要更改。

## [未发布]

## [0.9.0] - 2026-06-05

### 新增功能

- 引入 Dear ImGui 作为官方 UI 库，替代原 Win32 原生 UI
- 集成 DirectX11 作为 IMGUI 渲染后端
- 新增厅名称输入字段，与一麦字段分开
- 新增健康接档提示功能（档时超过6小时时弹出确认提示）
- 新增时间选择器对话框

### 改进

- UI 全面汉化
- 窗口尺寸调整为 1200x900，字体大小调整为 24px
- 厅信息区域布局调整，字段纵向排列
- 计数器区域布局调整，横向排列显示
- 主持数据复制格式改为"主持人名：拉新:X 二消:X 建联:X 三关:X"
- 档时默认值改为 00:00-03:00

### 修复

- 中文字体显示问题，加载项目内字体文件
- 主持数据区域数字不显示问题
- 数字输入框可输入负数问题
- 复制数据时中文乱码问题（使用 Unicode 编码）
- 健康提示确认后闪退问题
- 字体文件路径问题，改为 `data/font/` 目录结构

### 移除

- 原 Win32 原生 UI 代码

## [0.8.0] - 2026-06-04

### 新增功能

- 测试版构建的综合调试日志系统
- 调试控制台窗口支持（仅测试版）
- 正式版纯GUI模式（无控制台窗口）
- [INFO] 标签用于初始化和生命周期事件
- [TEST] 标签用于用户操作（按钮点击、数据更新）
- [ERROR] 标签用于错误情况
- 添加/删除主持时动态调整窗口高度
- 时间选择对话框支持鼠标滚轮
- 时间验证防止负时长
- 主持数据与汇总计数器实时同步

### 改进

- UI布局改为垂直排列
- 输入框左对齐以提高视觉一致性
- 改进窗口刷新机制减少闪烁
- 版本号更新为 0.8.0

### 修复

- 主持位置显示问题
- 按钮位置动态调整
- 文本显示问题（换行、斜杠位置）
- 窗口高度动态调整
- 缩放BUG
- 手动编辑主持数据不同步到汇总计数器

## [0.7.0] - 2026-06-03

### 新增功能

- 简单报表生成和导出

- 目标设定对话框
